Design Philosophy

3 Mantras

01
⚡

Instinctive

Design should be instinctive for the user without relying on instructions. If you need to explain it, redesign it.

02
↻

Iterative

There is no finish line to design. With each iteration, an opportunity emerges to get it better for the user. Ship, learn, improve.

03
🤝

Open

Be receptive to feedback from others, creating a collaborative and inclusive approach to design.
